# 🚀 Profile Card UI – Tony Stark

A modern and stylish Profile Card UI built using HTML & CSS. This project showcases a sleek glassmorphism design with vibrant gradient elements and a minimal layout.

---

## 🌐 Live Demo
https://profile-model.vercel.app/
---

## ✨ Features

- 🎨 Modern Glassmorphism UI Design
- 🌈 Gradient glowing background elements
- 👤 Profile card with avatar, name, and designation
- 🔘 Interactive buttons (Share Profile & Follow Me)
- 💡 Clean and responsive layout
- ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

- HTML5
- CSS3
---


## 🎨 UI Highlights

- Soft blurred background card
- Rounded corners with shadow
- Floating gradient circles for aesthetic appeal
- Centered layout for clean focus
